Installation

To install epipy, clone the repository and install using setup.py:

git clone https://github.com/cmrivers/epipy.git
python setup.py install

Quickstart

import epipy
import os
dir = os.path.dirname(__file__)

mers = pd.read_csv(os.path.join(dir, '../data/mers_line_list.csv'))
fig, ax = epipy.case_tree_plot(mers, 'Cluster ID', 'Case #', 'dates', 5, 4, 'Cluster ID')
